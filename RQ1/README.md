# Research Question 1: Hyperlink Prevalence in PR Descriptions

This folder contains the scripts and data relevant to **Research Question 1 (RQ1):** *How prevalent are hyperlinks in PR descriptions, and what is their distribution across different types of resources?*

## Overview
To answer this question, we conducted a preliminary study analyzing links found in pull request descriptions from the top 50 most-starred GitHub repositories. 

The complete dataset generated from this study is hosted on Zenodo and can be accessed here: 
📊 **[Dataset: Hyperlinks in PR Descriptions](https://zenodo.org/records/21343748)**

---

## Repository Contents

### Data Extraction & Processing
* **`check_link.py`** Extracts hyperlinks from the pull request descriptions for each repository.
* **`combine_all_repo.py`** Merges the extracted link data from all 50 repositories into a single consolidated CSV file.

### Data Analysis
* **`Min_max.py`** Calculates the minimum, maximum, and average number of links per pull request.
* **`check_media_type_github_type.py`** Calculates the percentage and proportion of internal GitHub links versus external links.
* **`check_github_type.py`** Analyzes and categorizes the specific types of links within the internal GitHub group.
* **`domain_analysis.py`** Analyzes the domains of the URLs within the external links group.
* **`quarter.py`** Calculates the distribution and amount of links across quartiles.

### Preparation for RQ2
* **`sample.py`** Performs stratified sampling on the dataset to prepare data for answering Research Question 2 (RQ2).
* **`700 Sample.csv`** The resulting dataset containing 700 stratified samples generated by `sample.py`, to be used for RQ2.
